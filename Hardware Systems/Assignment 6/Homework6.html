Q1 - Change Maker

Write an assembler program that reads in an integer number of pennies, and returns the equivalent in dollar coins, quarters, dimes, nickels and pennies, minimizing the number of coins. Continue looping until the end of file marker is reached.

The professor will provide an example program for your reference, runnable on Morbius.

You may assume that overflow and underflow do NOT occur.

Q2 - ASCII Table

Write an assembler program that reads two positive integers, low and high, and prints a table of conversions from low to high, printing each value in decimal, hexadecimal, binary, and as an ASCII character.

The professor will provide an example program for your reference, runnable on Morbius.

Use the operating system's kprintf() function to format your output.

Q3 - Least Common Multiple

Write an assembler program that reads two positive integers, x and y, and prints the Least Common Multiple of x and y. Continue looping until the end of file marker is reached.

The professor will provide an example program for your reference, runnable on Morbius.

You may assume that overflow and underflow do NOT occur.
